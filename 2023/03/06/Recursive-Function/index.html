Tail Recursion

Definition: The resursive statement is either the state value or the very last action performed.

A tail resursion can be converted into a while loop.

Murual Recursive

definition: When a recursive function is devided into two functions that call each other, the funtions are said to be mutually recursive.

  • a number is even if it is one more than an odd number
  • a number is odd if it is one more than an even number
